# Card Matching Game
Made in Unity 2021.3.45f2(LTS)

https://github.com/user-attachments/assets/db4bd750-26a2-4c08-8bcc-7874d40a8a3b

Support various card layouts, such as 2x2, 3x3, 5x6, etc. The cards should scale to fit the
target display area, whether it's the screen or another container widget.
Implemented a save/load system to persist progress between game restarts.
Included a scoring mechanism.
Integrated basic sound effects for card flipping, matching, mismatching, and game over
